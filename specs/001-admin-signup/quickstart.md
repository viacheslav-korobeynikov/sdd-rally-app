# Quickstart: Регистрация администратора ралли

**Branch**: `001-admin-signup`  
**Date**: 2025-12-08

## Предпосылки
- Go 1.21
- Docker + docker-compose
- PostgreSQL доступен через compose
- SMTP или тестовый mailhog для писем подтверждения

## Шаги локального запуска
1. Скопировать `.env.example` → `.env`, задать:
   - `POSTGRES_DSN`
   - `SMTP_HOST/PORT/USER/PASSWORD` или mailhog
   - `APP_BASE_URL` (для ссылок подтверждения)
2. Запустить инфраструктуру:
   ```bash
   docker-compose up -d postgres mailhog
   ```
3. Применить миграции:
   ```bash
   go run cmd/migrate/main.go up
   ```
4. Запустить сервер:
   ```bash
   go run cmd/server/main.go
   ```
5. Проверить регистрацию:
   - Открыть форму `/admin/signup`
   - Получить письмо в mailhog (если используется)
   - Перейти по ссылке подтверждения, затем войти.

## Тестирование
